ScreenName

@Target([AnnotationTarget.CLASS]) annotation class ScreenName

Used in the situations when the actual screen name is predefined, provided and static.

Should be used only on an Activity, a Fragment or any other variant of the same.

@ScreenName(value = AnalyticsScreenNames.MAIN_SCREEN)
class MainActivity : Activity() {
    ...
}
